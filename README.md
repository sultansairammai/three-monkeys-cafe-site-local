# ☕ Three Monkeys Café – Website Redesign Case Study

A modern, conversion-focused redesign of the Three Monkeys Café website, built as an interactive product experience with mobile-first UI/UX, chatbot integration, and dynamic user flow.

---

## 🌐 Live Demo

👉 https://sultansairamuk.github.io/three-monkeys-cafe-site-local/

---

## 🧠 Project Overview

This project reimagines the digital presence of **Three Monkeys Café** with a focus on:

- Enhancing first impressions  
- Improving navigation clarity  
- Creating a premium, interactive experience  
- Simulating a real-world ordering flow  

The goal was to transform a static café website into a **responsive, intelligent, and user-driven interface**.

---

## 🚨 Problem

Typical café websites often suffer from:

- Outdated or inconsistent design  
- Weak visual hierarchy  
- Limited engagement and interaction  
- Poor mobile experience  
- No clear ordering or booking flow  

This leads to **low user engagement and poor conversion**.

---

## 💡 Solution

Designed and developed a **mobile-first, interactive website** that:

- Uses structured navigation and CTA flow  
- Integrates an AI chatbot for engagement  
- Improves layout and readability  
- Simulates real product behavior (cart, interaction, flow)  

---

## ✨ Key Features

### 📱 Mobile-First Navigation
- Fixed bottom navigation bar (Home, Menu, Chat, Cart, Book)  
- Thumb-friendly UX  
- Smooth section transitions  

---

### 🤖 AI Chatbot System
- Interactive assistant  
- Multilingual support (English, Hindi, Telugu)  
- Helps users explore menu and combos  
- Integrated with cart and navigation  

---

### 🛒 Cart System
- Add/remove items dynamically  
- Real-time subtotal updates  
- Cart drawer interaction  
- Connected with chatbot behavior  

---

### 📐 UI/UX Enhancements
- Clean, modern layout  
- Improved spacing and hierarchy  
- Strong CTA placement  
- Better content structure  

---

### 📍 Layout Optimization
- Horizontal scroll “Good To Know” section  
- “Find Us” section repositioned  
- Improved visual flow
